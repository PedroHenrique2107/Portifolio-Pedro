# React + TypeScript + Vite

Projeto de portifolio com frontend em React.

## Contato sem backend (Web3Forms)

O formulario de contato envia direto para um servico externo, sem API propria.

### 1. Criar chave no Web3Forms

Acesse: https://web3forms.com/ e gere sua Access Key.

### 2. Configurar `.env`

Crie um arquivo `.env` na raiz do projeto:

```env
VITE_WEB3FORMS_ACCESS_KEY=YOUR_WEB3FORMS_ACCESS_KEY
```

### 3. Rodar o projeto

```bash
npm run dev
```
